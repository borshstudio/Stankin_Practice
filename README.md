# 📘 Стандарт WX
Компонент на C++, который получает текущую погоду и прогноз с сервиса OpenWeather

Этот проект — небольшой учебный компонент, который мы сделали для тренировки работы с внешними API. Он умеет отправлять запросы, разбирать JSON и выдавать аккуратные структуры с данными о погоде. Плюс есть простое CLI‑приложение, чтобы можно было посмотреть результат прямо в терминале.

## 🌤 Что умеет компонент
- Получение текущей погоды по названию города  
- Прогноз на ближайшие 24 часа  
- Обработка ошибок (неверный ключ, город не найден и т.д.)  
- Лёгкое кэширование, чтобы не обращаться к API лишний раз  
- Небольшая консольная утилита `weather_cli`
